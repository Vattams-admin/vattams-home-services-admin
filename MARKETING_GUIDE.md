# VATTAMS Home Services - Marketing Guide

## Overview
This guide covers all digital marketing features available in VATTAMS Home Services v3.0.

## Google Business Profile
1. Navigate to Admin > Google Business
2. Fill in: business name, description, address, service areas, working hours, contact numbers, WhatsApp, email, website
3. Add Google Maps link, Google Business Profile link, and Google Review link
4. Customers can view your business on Google Maps, get directions, and leave reviews

## Google Analytics (GA4)
1. Navigate to Admin > Analytics & Pixel
2. Enter your GA4 Measurement ID and API Secret
3. The system tracks: page views, visitors, sessions, bookings, revenue, WhatsApp clicks, call clicks, service views
4. View event breakdown and recent activity in the same page
5. Export event data as CSV

## Meta Pixel (Facebook)
1. Navigate to Admin > Analytics & Pixel
2. Enter your Meta Pixel ID and Access Token
3. Tracked events: Lead, Booking, WhatsApp Click, Call Click, Contact Form, Page Views

## Google Search Console
1. Navigate to Admin > Analytics & Pixel
2. Enter your Google Search Console verification token
3. sitemap.xml and robots.txt are pre-configured
4. Blog posts support canonical URLs, meta titles, and meta descriptions

## Social Media Management
1. Navigate to Admin > Settings > Social Media section
2. Add URLs for: Facebook, Instagram, YouTube, LinkedIn, X (Twitter), Telegram, WhatsApp Business
3. Social icons appear in the website header and footer
4. Navigate to Admin > Marketing > Social Links tab for additional social link management

## Marketing Dashboard
1. Navigate to Admin > Marketing Dashboard
2. View: website visitors, total bookings, revenue, WhatsApp clicks, call clicks
3. Most viewed services (top 5)
4. Top districts by booking volume
5. Campaign performance with impressions, clicks, conversions
6. Export PDF and CSV reports

## Promotions
### Banner Manager
1. Navigate to Admin > Marketing > Banners tab
2. Create banners with: title, image URL, link URL, position (hero/midpage/footer)
3. Track click counts per banner
4. Toggle active/inactive

### Popup Offers
1. Navigate to Admin > Marketing > Popups tab
2. Create popups with: title, content, type (popup/announcement/promotion), start/end dates
3. Set as dismissible or non-dismissible

### Campaigns
1. Navigate to Admin > Marketing > Campaigns tab
2. Create campaigns: name, type (festival/seasonal/first_booking/referral/general), channel (whatsapp/push/email/social)
3. Track campaign status: draft/active/paused/completed
4. View campaign metrics in Marketing Dashboard

## Email Marketing
1. Navigate to Admin > Email Marketing
2. **Campaigns tab**: Create email campaigns with subject, template, recipient type (all/customers/technicians)
3. **Templates tab**: Create reusable templates (newsletter, booking reminder, service reminder, promotional)
4. Track sent count, open count, click count per campaign

## Blog CMS
1. Navigate to Admin > Blog CMS
2. **Posts tab**: Create blog posts with SEO fields (meta title, meta description, canonical URL), featured images, categories, tags
3. **Categories tab**: Organize posts by category with auto-generated slugs
4. **Tags tab**: Add tags for better content discovery
5. Blog posts appear on the public /blog page with search functionality

## Customer Reviews
1. Navigate to Admin > Reviews
2. View all reviews with rating distribution chart
3. Approve/feature reviews for display on public /reviews page
4. Add reviews manually (e.g., Google reviews imported)
5. Export reviews as CSV
6. Featured reviews appear on the homepage

## Coupons & Offers
1. Navigate to Admin > Coupons & Offers
2. Create coupons with 6 offer types: flat, percentage, festival, seasonal, first_booking, referral
3. Target by district or service category
4. Set usage limits and expiry dates
5. View coupon reports and export CSV

## Referral Program
1. Navigate to Admin > Referrals
2. View referral analytics: total, successful, pending, rewards paid
3. Manage reward status (pending -> credited/rejected)
4. View top referrers leaderboard
5. Export referral data as CSV

## CRM
1. Navigate to Admin > CRM
2. Search customers and view their complete timeline
3. Add notes, schedule follow-ups
4. Track complaints, service reminders, AMC renewals
5. View Customer Lifetime Value and Satisfaction Score per customer
