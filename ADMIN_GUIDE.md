# VATTAMS Home Services - Admin Guide

## Getting Started
1. Login at `/login` with your admin email and password
2. You will be redirected to `/admin/dashboard`

## Dashboard
- View stats: Total Customers, Technicians, Pending Verifications, Active Bookings, Total Revenue
- Approve or reject pending technician applications
- View recent bookings

## Technician Verification
1. Navigate to **Verification** in the sidebar
2. Filter by status: Pending Registration, Fee Pending, Under Review, Approved, Rejected, Suspended
3. For each technician: View Documents, View Profile, Approve, Reject, Suspend, Activate
4. **Refund Management**: For technicians with 4+ completed jobs, process verification fee refunds

## CRM Module
1. Navigate to **CRM** in the sidebar
2. Search customers by name, email, or mobile
3. For each customer: View Timeline, Add Note, Schedule Follow-up, View Complaints, View Service Reminders, View AMC Renewals
4. View Customer Satisfaction Score and Lifetime Value

## Coupons & Offers
1. Navigate to **Coupons & Offers** in the sidebar
2. Create coupons with offer types: Flat, Percentage, Festival, Seasonal, First Booking, Referral
3. Target by district or service category, set usage limits and expiry
4. View coupon reports and export CSV

## Referral System
1. Navigate to **Referrals** in the sidebar
2. View stats: Total Referrals, Successful, Pending, Total Rewards
3. Manage reward status, view top referrers leaderboard
4. Export referral data as CSV

## Marketing Module
1. Navigate to **Marketing** in the sidebar
2. **Banners**: Add/edit homepage banners with position and click tracking
3. **Popups**: Create popups, announcements, promotions with date ranges
4. **Campaigns**: Create promotional campaigns (festival, seasonal, first booking, referral)
5. **WhatsApp Templates**: Manage message templates for campaigns
6. **Social Links**: Manage social media and YouTube links
7. **Analytics**: View campaign performance, impressions, clicks, conversions

## Marketing Dashboard
1. Navigate to **Marketing Dashboard** in the sidebar
2. View: Website Visitors, Total Bookings, Revenue, WhatsApp Clicks, Call Clicks
3. Most Viewed Services and Top Districts charts
4. Campaign Performance table with metrics
5. Export PDF and CSV reports

## Google Business Profile
1. Navigate to **Google Business** in the sidebar
2. Manage: business name, description, address, service areas, working hours, contacts, WhatsApp, email, website
3. Add Google Maps link, Business Profile link, and Review link
4. Customers can view on Google Maps, get directions, leave reviews

## Analytics & Pixel
1. Navigate to **Analytics & Pixel** in the sidebar
2. Configure GA4 (Measurement ID, API Secret)
3. Configure Meta Pixel (Pixel ID, Access Token)
4. Configure Google Search Console (verification token)
5. Configure Firebase (JSON config)
6. View analytics events: visitors, page views, WhatsApp clicks, call clicks, booking events
7. Export event data as CSV

## Email Marketing
1. Navigate to **Email Marketing** in the sidebar
2. **Campaigns**: Create and track email campaigns (newsletter, booking reminder, service reminder, promotional)
3. **Templates**: Create reusable email templates by category

## Blog CMS
1. Navigate to **Blog CMS** in the sidebar
2. **Posts**: Create SEO-friendly blog posts with meta tags, featured images, categories, tags
3. **Categories**: Organize content with auto-generated slugs
4. **Tags**: Add tags for content discovery

## Customer Reviews
1. Navigate to **Reviews** in the sidebar
2. View all reviews with rating distribution chart
3. Approve, feature, or delete reviews
4. Add reviews manually
5. Export reviews as CSV

## AI Assistant
1. Navigate to **AI Assistant** in the sidebar
2. Use quick suggestions: Booking Insights, Revenue Summary, Technician Performance, Customer Trends, Suggested Actions
3. Ask questions in natural language - the assistant fetches real data for context-aware responses

## Business Settings
1. Navigate to **Settings** in the sidebar
2. **Company Info**: Name, logo, email, website
3. **Contact**: Support phones, WhatsApp numbers (primary + technician)
4. **Business**: Working hours, UPI ID, invoice prefix
5. **Theme**: Primary and accent colors
6. **Language**: English or Tamil
7. **Social Media**: Facebook, Instagram, YouTube, LinkedIn, X, Telegram, WhatsApp Business URLs
8. **Google**: Maps link, Business Profile link, Review link, GA4 ID, Meta Pixel ID, Site Verification
9. **Holiday Calendar**: Add holidays (recurring or one-time)
10. **Service Pricing**: Manage service prices by category

## Revenue & Reports
1. **Revenue**: View revenue by type, filter by time period, export PDF/CSV
2. **Reports**: View booking stats, completion rates, top services/technicians, monthly revenue charts

## Audit Logs
- View all admin actions with timestamps
- Search by action type or entity type
