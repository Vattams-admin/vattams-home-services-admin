# VATTAMS Home Services

Professional home services platform for Tamil Nadu. AC service, washing machine repair, plumbing, electrical, and more.

## Tech Stack
- React 18 + TypeScript + Vite 5
- Tailwind CSS v4
- Supabase (PostgreSQL, Auth, Storage, RLS)
- jsPDF + qrcode for invoice generation

## Features
- Technician verification system with ₹50 activation fee
- Wallet with balance, earnings, and refund tracking
- Refund eligibility after 4 successful jobs
- Admin verification module with document review
- Revenue management with daily/weekly/monthly/yearly analytics
- Customer booking flow with live tracking
- Invoice generation with UPI QR codes
- Coupon/discount management
- Audit logging for admin actions
- PWA support with manifest
- SEO (robots.txt, sitemap.xml, Open Graph)
- WhatsApp & YouTube integration
- Bilingual support (English/Tamil)
- Role-based access control (customer/technician/admin/super_admin)
- PDF/CSV report export
